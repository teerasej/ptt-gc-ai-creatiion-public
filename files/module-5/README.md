# Module 5 Support Files

ไฟล์ใน Module 5 ใช้สำหรับ Agent ตัวอย่างชื่อ `PTT GC Procurement Policy Assistant` โดยผู้เรียนทุกคนใช้ scenario เดียวกัน เพื่อให้ฝึกเรื่อง UX, publishing, analytics และ pilot readiness ได้บนฐานเดียวกัน

## Expected Knowledge Files

- `Procurement Policy Handbook.pdf`
- `Vendor Onboarding Checklist.docx`
- `Purchase Approval Matrix.xlsx`

## ใช้ไฟล์เหล่านี้ตอนไหน

- Exercise 1: อัปโหลดทั้ง 3 ไฟล์เป็น knowledge sources ของ Agent ใหม่
- Exercise 2: ใช้ Agent เดิมที่มี knowledge ครบแล้วเพื่อ publish และซ้อม demo
- Exercise 3: ใช้ Agent เดิมเพื่อเก็บ published conversations สำหรับ Analytics
- Exercise 4: ใช้หลักฐานจาก knowledge, publish status และ analytics เพื่อทำ Pilot Decision Pack

## Notes

- ไฟล์ทั้งหมดในโฟลเดอร์นี้เป็นข้อมูลจำลองเพื่อการฝึกอบรม
- ให้ใช้ไฟล์จากโฟลเดอร์ `files/module-5/` เป็นหลักสำหรับ Module 5
- ห้ามอัปโหลด procurement policy จริงหรือข้อมูล vendor จริงลงใน Agent

> ⚠️ Note: หากห้องเรียนยังไม่ได้รับไฟล์จริงจากผู้สอน ให้ใช้ README นี้เป็น checklist ของไฟล์ที่ต้องเตรียมก่อนเริ่ม Exercise 1
